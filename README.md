внедрение зависимостей (DI) через Koin.

okhttp — HTTP‑клиент для запросов.

retrofit — типобезопасный HTTP‑клиент для REST API.

converter.kotlinx.serialization и converter.gson — конвертация JSON в объекты Kotlin (два варианта: Kotlin Serialization и Gson).

room.ktx — ORM для работы с SQLite (локальная база данных).

ksp(room.compiler) — генерация кода для Room через KSP (быстрее, чем kapt).

coil.compose — загрузка и отображение изображений в Compose.

coil.network.okhttp — интеграция Coil с OkHttp (кеширование, оптимизация запросов).
